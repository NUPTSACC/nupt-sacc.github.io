# 网页设计大赛 #
## SACC blog ##
-------
### 1.主要特点 ###
我们组的网页可以根据屏幕的大小做到自适应，保证不会在其他分辨率的电子设备上排版错乱。

### 2.主页板块介绍 ###
主页总共分为5个板块，网页顶端的导航栏及媒体介绍、网页中部分为三个模块，分别是社团的总体介绍、近期活动介绍（原意是做成社团发展史的时间轴，但是由于素材原因，就先暂定为社团活动介绍）和部门介绍，网页底部是页面声明和广告。

### 3.所用技术 ###
HTML CSS Javascript
JQuery Bootstrap iScroll
